## Style Element
Inside _head_ element <br>
Can set _properties_ <br>
Example of a _type selector_: <br>
```
element {
 property: value;
}
```

Can group selectors: <br>
```
selector1, selector2 {
  property: value;
}
```

Best practice is to keep styles in separate files. <br>
Example _style.css_ <br>
